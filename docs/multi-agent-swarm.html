Swarm

Agents decide at runtime who handles the next step. Execution begins at an entry point and flows through handoffs until a final answer is produced.

When to Use Swarm

  • The routing logic depends on the content of each response
  • You cannot predict the sequence of agents ahead of time
  • Tasks are open-ended and may need different specialists depending on what comes up
  • You want the model itself to decide who is best suited for each step

How It Works

Each agent in the swarm is a SwarmMember with an ID and a description. The swarm exposes all other members' descriptions to the active agent via a hidden handoff tool. When an agent decides it has finished its part, it hands off to another member by name.

User Input → triage → handoff → specialist
→ handoff → writer → Final Response

Basic Example

Triage → Specialist → WriterSwift
import StrandsAgents

let triageAgent = Agent(
    model: provider,
    systemPrompt: """
    You triage incoming requests and hand off to the right specialist.
    Available specialists:
    - "swift-expert": Swift language and SDK questions
    - "cloud-expert": AWS, cloud infrastructure, networking
    - "writer": Turning technical content into clear prose
    Hand off immediately — do not try to answer yourself.
    """
)

let swiftExpert = Agent(
    model: provider,
    systemPrompt: "You are a Swift language expert. Answer in depth with code examples. Hand off to 'writer' when done."
)

let cloudExpert = Agent(
    model: provider,
    systemPrompt: "You are an AWS and cloud infrastructure expert. Give precise, actionable answers. Hand off to 'writer' when done."
)

let writerAgent = Agent(
    model: provider,
    systemPrompt: "You take expert technical content and rewrite it as clear, well-structured prose for developers."
)

let swarm = SwarmOrchestrator(
    members: [
        SwarmMember(id: "triage",       description: "Routes requests to the right specialist", agent: triageAgent),
        SwarmMember(id: "swift-expert",  description: "Answers Swift language questions",         agent: swiftExpert),
        SwarmMember(id: "cloud-expert",  description: "Answers cloud and AWS questions",          agent: cloudExpert),
        SwarmMember(id: "writer",        description: "Writes the final polished response",        agent: writerAgent),
    ],
    entryPoint: "triage"
)

let result = try await swarm.run("How do I deploy a Swift Vapor app on AWS Lambda?")
print(result.output)

Handoffs

A handoff transfers execution to another agent, carrying the full conversation context. The receiving agent sees everything that happened before — all previous messages, tool results, and handoff notes.

ℹ️

Agents must be explicitly instructed in their systemPrompt to hand off and to whom. The model will not hand off unless told it should. Keep the list of available agents and their roles in the system prompt.

Entry Point

The entryPoint parameter specifies which agent receives the user's initial message. Usually this is a triage or router agent that assesses the task and delegates, but it can be any member:

Swift
// Start directly at a specialist for focused tasks
let swarm = SwarmOrchestrator(
    members: [
        SwarmMember(id: "analyst", description: "Analyzes data",   agent: analystAgent),
        SwarmMember(id: "writer",  description: "Writes reports",  agent: writerAgent),
    ],
    entryPoint: "analyst"  // skip triage — always start with the analyst
)

Real-World Example: Customer Support Swarm

Support triage systemSwift
let triageBot = Agent(
    model: provider,
    systemPrompt: """
    You are a support triage agent. Classify the user's request and hand off:
    - Billing questions → "billing-agent"
    - Technical issues → "technical-agent"
    - Account questions → "account-agent"
    Do not answer. Only triage and hand off.
    """
)

let billingAgent = Agent(
    model: provider,
    tools: [lookupInvoice, issueRefund],
    systemPrompt: "You handle billing issues. You have tools to look up invoices and issue refunds."
)

let technicalAgent = Agent(
    model: provider,
    tools: [searchDocs, checkSystemStatus],
    systemPrompt: "You resolve technical issues. Search the docs and check system status to diagnose problems."
)

let accountAgent = Agent(
    model: provider,
    tools: [lookupAccount, resetPassword],
    systemPrompt: "You handle account questions — password resets, profile updates, account status."
)

let swarm = SwarmOrchestrator(
    members: [
        SwarmMember(id: "triage",    description: "Classifies and routes support requests", agent: triageBot),
        SwarmMember(id: "billing",   description: "Handles billing and payment issues",      agent: billingAgent),
        SwarmMember(id: "technical", description: "Resolves technical problems",             agent: technicalAgent),
        SwarmMember(id: "account",   description: "Manages account settings",                agent: accountAgent),
    ],
    entryPoint: "triage"
)

let result = try await swarm.run("I was charged twice for my subscription last month")
print(result.output)
