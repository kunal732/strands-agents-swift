# Examples

Example projects built with the Strands Agents Swift SDK. Each lives in its own repo so it can be cloned and run independently.

| Example | Description |
|---------|-------------|
| [Strands CLI](https://github.com/kunal732/strands-cli-example) | Minimal command-line agent with local MLX inference and tool calling. Good starting point for understanding the SDK. |
| [Desktop Assistant](https://github.com/kunal732/strands-desktop-assistant) | macOS menu bar app that controls your desktop via MCP tools. Uses Claude Sonnet on Bedrock for reasoning and a local MCP server for automation (clicking, typing, window management). |
